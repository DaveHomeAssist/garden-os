# Garden OS Game Timeline Package

This package scaffolds delivery for the three game modes discussed:
- Plant Strategy Simulator
- Rescue Mission: Pest Outbreak
- Garden League Draft

## Files
- `ROADMAP_0-180_DAYS.md` - phased plan with sequence and dependencies
- `GAME_MODE_SCOPES.md` - mode scope, loops, must-have vs nice-to-have
- `SPRINT_BOARD_TEMPLATE.md` - reusable sprint planning template
- `SPRINT_CALENDAR_12.md` - 24-week scaffold using 12 two-week sprints
- `METRICS_AND_LAUNCH_GATES.md` - release quality gates and KPI targets
- `GARDEN_LEAGUE_COMMUNITY_HUB.md` - community operations scaffold for league seasons

## How to Use
1. Start each planning cycle from `ROADMAP_0-180_DAYS.md`.
2. Pull sprint items from `GAME_MODE_SCOPES.md`.
3. Run weekly tracking using `SPRINT_BOARD_TEMPLATE.md`.
4. Enforce ship quality using `METRICS_AND_LAUNCH_GATES.md`.

## Recommended Build Order
1. Plant Strategy Simulator
2. Rescue Mission: Pest Outbreak
3. Garden League Draft
