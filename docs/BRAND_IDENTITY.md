# VISUAL IDENTITY SYSTEM: "Garden OS"
**CORE AESTHETIC DIRECTION**

* Think:
  ```
  A sun-warmed field journal left open on a potting bench, crossed
  with the control panel of a greenhouse climate system.
  Handwritten seed packet labels next to precise soil pH readouts.
  Warm, grounded, deeply personal. Built for the backyard, not the app store.
  ```

  ```
  A grandmother's recipe box reimagined as garden planning software.
  Every element should feel like it was designed to survive
  dirt under the fingernails, morning dew, and a full growing season.
  ```

* Key adjectives:
  ```
  Homegrown. Sun-warmed. Grounded. Sanctuary energy.
  Patient precision. Kitchen-table authority.
  ```

  ```
  Not "gardening app branding." Not Miracle-Gro catalog. Not Silicon Valley ag-tech.
  More: seed library index card system. Grandma's almanac with margin notes.
  Community garden bulletin board with hand-drawn plot maps.
  ```

* Reference anchors:
  ```
  - Vintage seed packet illustrations (warm watercolor, hand-lettered variety names)
  - Farmer's almanac typography and layout (dense, trusted, seasonal)
  - USDA plant hardiness zone maps (earthy gradients, contour regions)
  - Victory Garden propaganda posters (bold type, warm patriotic earth tones)
  - Beatrix Potter's garden illustrations — ink and soft watercolor, never photographic
  - Hand-drawn raised bed blueprints on graph paper, pencil and colored pencil
  ```

---

**TYPOGRAPHY FEELING**

* Primary display:
  ```
  Fraunces — "seed packet energy."
  Warm, slightly quirky optical serif with organic curves.
  The kind of typeface that looks like it was designed by someone
  who spends weekends at the farmer's market.
  Weight 600 for titles, 300 for elegant body.
  Soft, approachable, never intimidating.
  ```

* Secondary UI:
  ```
  DM Sans — "potting bench clarity."
  Clean and friendly without being clinical. Reads well at small sizes
  on a phone screen held with dirty gloves. The practical workhorse
  that lets Fraunces be the personality.
  Weight 400 for body, 500 for emphasis.
  ```

* System / data:
  ```
  DM Mono — "greenhouse instrument readout."
  Every number (score, zone, days-to-harvest, cell position) should feel
  measured and intentional. Small caps for section labels. Tight tracking
  for grid coordinates. The monospace creates just enough "system" feel
  to remind you there's real logic behind the warmth.
  ```

* Notes:
  ```
  Never use cold geometric sans-serifs (no Roboto, no SF Pro, no Helvetica).
  The combination of warm serif + friendly sans + precise mono creates
  the exact tension between "grandmother's garden wisdom" and
  "deterministic scoring engine" that defines Garden OS.
  ```

## [IMAGE 3 — LOGO / SYMBOLIC MARK]

* Mark concept:
  ```
  A top-down view of a raised bed garden — a rounded rectangle (the bed
  frame) divided into an 8x4 grid of cells. The cells are filled with
  small, abstracted crop symbols in varying earth-tone and green colors
  (leaf shapes, circles for tomatoes, tall lines for climbing beans).

  Above the bed: a simple arch shape representing the cattle panel
  trellis structure, with a tiny climbing vine silhouette.

  At the center of the grid: a single golden sun dot (#e8c84a) with
  a subtle radial glow — the scoring engine's "eye," the intelligence
  behind the layout.

  Below the bed frame: no compass blade, no weapon metaphor.
  Instead, three small root lines extending down — organic, alive,
  growing into the ground. The mark is literally rooted.
  ```

* Geometry rules:
  ```
  Bed frame is a rounded rectangle — proportioned 2:1 (8x4 ratio).
  Grid lines: thin, 0.5px, suggesting cells without dominating.
  Crop symbols: simplified to 2-3px geometric marks (circles, diamonds,
  vertical lines). No detailed illustration — abstraction only.
  Trellis arch: single smooth curve above the bed, open at bottom.
  Root lines: 3, organic curves extending 30% of bed height below.
  Sun dot: centered, 4px, golden, with a 1px radial ring at 50% opacity.
  The mark must read at 16px (favicon) — at that size, show only the
  rounded rectangle with the golden center dot and trellis arch.
  ```

* Wordmark:
  ```
  GARDEN OS — Fraunces SemiBold, all caps, letter-spacing: 0.15em.
  Sized to match the bed width optically.
  Color: soil brown (#5c3d1e) on light, cream (#f7f2ea) on dark.
  The "OS" is slightly lighter weight (Fraunces 300) to create a
  subtle hierarchy — GARDEN is the soul, OS is the system.
  ```

* Tagline:
  ```
  MOM'S BACKYARD SCORING ENGINE
  DM Mono, 8-9px, letter-spacing: 0.16em, all caps.
  Color: muted warm brown (#8a7b6b) — secondary, never competing.
  Alternate tagline for formal contexts: RAISED BED PLANNING SYSTEM
  ```

* Color rules:
  ```
  Mark on light backgrounds: soil brown (#5c3d1e) outlines, leaf green
  (#3d7a4f) crop symbols, golden (#e8c84a) center dot.
  Mark on dark backgrounds: cream (#f7f2ea) outlines, leaf-bright
  (#5aab6b) crop symbols, golden center dot.
  The golden sun dot is the ONLY accent color that appears in BOTH
  variants. Green never appears in the wordmark — only in the mark.
  ```

* Texture rules:
  ```
  Subtle watercolor grain overlay at low opacity (5-8%) on the crop
  fill areas. Grid lines are clean and precise (the system underneath).
  This tension — organic fills inside precise geometry — IS the brand.
  No gradients except the sun dot radial glow. Flat, reproducible.
  ```

* Scalability notes:
  ```
  Favicon (16px): rounded rect + trellis arch + golden center dot only.
  Small (32-64px): bed frame + arch + center dot + hint of grid.
  Medium (64-128px): full mark minus tagline.
  Large (128px+): full lockup with wordmark and tagline.
  ```

## CAST OF CHARACTERS

Garden OS has a multi-voice personality system. These characters appear in the
Season Engine game and can be used in UI toast messages, onboarding, and error states.

### Garden GURL (Primary Voice)
```
Role: Enthusiastic mentor, main guide, narrator
Tone: Warm, encouraging, slightly breathless with excitement
  about what's growing. Uses exclamation marks sincerely.
  "Your tomatoes scored a 9.2 — that's the best adjacency
  bonus I've seen all season!"
Visual: If illustrated, wears the Garden OS apron,
  always has dirt on their hands, perpetually delighted.
When: Good news, tutorials, scoring explanations, celebrations.
```

### Onion Man (Secondary Voice)
```
Role: Wry pragmatist, reality checker, weather warning system
Tone: Dry, practical, affectionate pessimism. Talks about root depth
  and drainage like they're discussing philosophy.
  "Look, I'm not saying your cucumber placement is bad.
  I'm saying the drainage coefficient says what it says."
Visual: Stout, wearing a flat cap and muddy boots.
  Perpetually squinting at the sky.
When: Warnings, practical advice, weather events, scoring penalties.
```

### Vegeman (Tertiary Voice)
```
Role: Earnest educator, crop database walking
Tone: Enthusiastic but precise. The friend who actually read the
  seed packet instructions and wants to tell you about them.
  "Fun fact — basil and tomatoes aren't just kitchen companions.
  Their root exudates actually improve each other's soil uptake!"
Visual: Tall, thin, wearing a sun hat too large for their head.
  Always carrying a reference book.
When: Crop facts, adjacency explanations, educational moments.
```

### Garden Critters (Background Voice)
```
Role: Playful antagonists, event narrators
Tone: Mischievous, chaotic, not malicious. The squirrel that dug up
  your bulbs is sorry — but only a little.
  "The rabbits have discovered the lettuce row.
  They seem... very enthusiastic about it."
Visual: Illustrated as simple, slightly cartoonish animals —
  rabbits, squirrels, birds, the occasional deer silhouette.
When: Random events, pest encounters, weather chaos, harvest theft.
```

## ORIGIN STORY

```
Garden OS began in a casual Claude conversation when Dave asked for help
laying out his mom's original 4x4 raised bed with cucumbers, leaf lettuce,
and bush green beans. Claude created an interactive visual planner artifact.

When Dave said "move the garden thread into a separate chat," he started
a new conversation — and that conversation became the seed for the entire
ecosystem. The 4x4 became 8x4. The planner gained scoring. The scoring
gained a visualizer. The visualizer needed a fairness tester. A game
emerged to teach the scoring through narrative. Build and ops guides
followed to document the real physical structure.

Every tool in Garden OS traces back to three crops in a 4x4 bed.
The project is a love letter from a son to his mother's garden,
built one Claude conversation at a time.
```
