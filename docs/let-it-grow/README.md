# Garden OS — Let It Grow Documentation Package

Documentation staging folder containing 20 files organized across seven categories for the Garden OS open-world expansion: **Let It Grow**.

## Structure

| Directory | Contents |
|-----------|----------|
| `00-overview/` | Project overview, current state audit |
| `10-product/` | Expansion brief, naming/franchise structure |
| `20-game-design/` | Game design document, skills/items/economy, NPC/quest framework |
| `30-systems/` | Garden grid, intervention system, scoring algorithm, time/seasons/events |
| `40-world-narrative/` | Chapter structure, characters/cutscenes, open world transition plan |
| `50-technical/` | Source file map, rendering architecture, save state/data model |
| `60-roadmap/` | Phase plan, Unity port evaluation, modernization roadmap |

## Origin

These specs were originally staged in the `theme-organizer-resource` repo (branch `claude/let-it-grow-spec-jSmqH`) and have been moved here to their correct home in the Garden OS project.

## Relationship to Existing Specs

The existing `specs/` directory contains the canonical specifications for Garden OS Story Mode (scoring rules, season engine, crop data, etc.). This `docs/let-it-grow/` package extends those specs with the open-world expansion design — it references but does not replace the canonical spec files.
