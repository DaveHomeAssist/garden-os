# Garden OS: Concept Board Cohesion

This document captures the intended visual cohesion across character art, environment style, UI, event cards, chapter art, and marketing surfaces.

## Core Thesis
Garden OS should read as **print-filtered Philly realism**: worn surfaces, seasonal honesty, and grounded neighborhood detail, with bold character overlays that interpret the world rather than replace it.

## 1) Art Pillars
1. **Worn Surface, Living Line**
   - Variable-weight ink lines over tangible materials (paper grain, cedar wear, soil texture).
   - Avoid pristine vector-clean rendering.
2. **Philly Grit as Frame**
   - Brick, chain-link, alley concrete, visible infrastructure.
   - Garden color and life should pop against urban wear.
3. **Seasonal Honesty**
   - Season shifts are structural: light angle, stress, saturation, and texture changes.
   - Not cosmetic palette swaps.
4. **Authored, Not Algorithmic**
   - Deliberate asymmetry, selective imperfection, controlled print artifacts.

## 2) Color Logic by Season
- **Early Spring:** cold ground, pale growth, pewter sky.
- **Late Spring:** warmer ground, brighter growth accents, clearer contrast.
- **Summer:** baked earth, dense greens, hazy highlights.
- **Late Summer / Early Fall:** dusty neutrals, fractured amber/yellow greens.
- **Fall:** wet brick, dry stalk browns, low dramatic light.
- **Winter:** muddy/frozen substrate, minimal growth chroma, blue-grey ambient cast.

Design rule: keep ground muted and let meaningful growth saturation be earned.

## 3) Texture Language
Primary materials:
- Cedar plank grain (weathered, repaired)
- Soil body (damp vs compacted vs dry)
- Concrete/asphalt wear
- Rusted/weathered metal (trellis, chain-link)

Execution rule: avoid smooth synthetic gradients as the dominant surface treatment.

## 4) Character-to-World Balance
- Characters remain graphic and iconic.
- Environment remains observed and materially grounded.
- Characters should mostly appear through commentary surfaces (panels/cards/overlays), not as physically integrated mascot actors in-world.

## 5) UI Integration
UI should use journal/almanac framing with tactical clarity:
- Bordered paper-like panels
- Utility typography hierarchy
- Print-inspired overlays (halftone, stamp, margin notes)
- Strong readability first, style second

## 6) Anti-Childlike Guardrails
Avoid:
- candy palettes
- bubbly UI geometry
- toy-like iconography
- over-cute expression language

Prefer:
- angular restraint
- editorial framing
- tactical overlays
- adult emotional register

## 7) Screenshot Signature Checklist
A screenshot should include at least 3 of:
- unmistakable 8x4 bed geometry
- urban Philly context fragment (brick/fence/alley/concrete)
- visible strategic overlay (support/access/stress)
- one strong character commentary insert
- season-specific light/mood state

## 8) Visual Direction Decision
Recommended direction:
**Zine Press foundation** with borrowing from:
- Almanac framing for UI structure
- Backyard observed realism for place credibility

Why:
- strongest authored identity
- strongest awards/store/trailer coherence
- consistent with anti-cozy positioning

## 9) Production Notes
- Build a source-of-truth asset manifest for all concept-board references.
- Add a licensing status column for every external reference image.
- Treat external links as reference-only unless rights are cleared.

## 10) Cohesion Acceptance Criteria
Art direction is "coherent" when:
- characters, UI, and environment all share the same print-material logic,
- season mood shifts are visible without breaking system readability,
- screenshots are recognizable as Garden OS without logo dependency,
- the game reads as grounded strategy, not cozy toy simulation.
